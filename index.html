ML Evaluation Cheatsheet

\(FNR = \frac{F_n}{F_n + T_p}\)

\(FPR = \frac{F_p}{F_p + T_n}\)

\(TPR = \frac{T_p}{T_p + F_n}\)

\(TNR = \frac{T_n}{T_n + F_p}\)

\(P = \frac{T_p}{T_p + F_p}\)

\(R = \frac{T_p}{T_p + F_n}\)

\(F_1 = 2\frac{P \times R}{P + R}\)

\(Accuracy = \frac{T_p + T_n}{T_p + T_n + F_p + F_n}\)
